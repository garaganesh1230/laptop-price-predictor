# 💻 Laptop Price Predictor

A Machine Learning web application that predicts laptop prices based on hardware specifications.

## 🚀 Features

- Predict laptop prices instantly
- User-friendly Streamlit interface
- Supports multiple laptop brands
- Dynamic price category detection
- Feature impact visualization
- Real-time prediction

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Pickle

## 📊 Input Features

- Company
- Laptop Type
- RAM
- SSD Storage
- HDD Storage
- CPU Brand
- CPU Type
- GPU Brand
- Operating System
- Screen Size
- IPS Display
- PPI
- Weight

## 🎯 Prediction Categories

- 💸 Budget Laptop
- ⚖️ Mid-Range Laptop
- 💎 Premium Laptop

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
